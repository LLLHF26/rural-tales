## 通用说明

### 基础信息

| 项目 | 说明 |
|------|------|
| **Base URL** | `https://api.example.com/v1/admin` |
| **请求格式** | JSON |
| **响应格式** | JSON |
| **认证方式** | JWT Token，Header：`Authorization: Bearer <token>` |
| **权限** | 所有接口需管理员身份，除 `/auth/login` 外 |

### 通用响应结构

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

| code | 含义 |
|------|------|
| 0 | 成功 |
| 1001 | 参数错误 |
| 1002 | 未登录 / Token 过期 |
| 1003 | 无管理员权限 |
| 2001 | 数据不存在 |
| 2002 | 操作冲突 |
| 5000 | 服务器内部错误 |

## 五、AI 生成模块

### 5.1 AI 生成剧本（基于已有乡村）

根据数据库中已有的乡村文化素材，AI 自动生成完整剧本框架。

```
POST /ai/generate-script
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| villageId | int | 是 | 目标乡村 ID |
| type | string | 是 | 剧本类型：mystery / history / family |
| difficulty | int | 是 | 难度 1-5 |
| estimatedDuration | int | 是 | 预计时长（分钟） |
| extraRequirement | string | 否 | 补充需求（如"融入当地茶文化"） |

---

### 5.1.1 AI 生成剧本（纯文本素材）★新增

不依赖数据库乡村记录，直接提交文本素材生成剧本。适用于测试调试、快速原型场景。

```
POST /ai/generate-script-from-text
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| villageName | string | 是 | 村庄名称 |
| description | string | 否 | 村庄简介 |
| cultures | string | 否 | 文化素材文本（每行一条：类型\|标题\|内容） |
| spots | string | 否 | 实景地点文本（每行一条：名称\|纬度\|经度\|描述） |
| type | string | 是 | 剧本类型：mystery / history / family |
| difficulty | int | 是 | 难度 1-5 |
| estimatedDuration | int | 是 | 预计时长（分钟） |
