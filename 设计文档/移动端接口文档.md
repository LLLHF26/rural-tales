## 通用说明

### 基础信息

| 项目 | 说明 |
|------|------|
| **Base URL** | `https://api.example.com/v1` |
| **请求格式** | JSON（`Content-Type: application/json`） |
| **响应格式** | JSON |
| **认证方式** | JWT Token，Header：`Authorization: Bearer <token>` |
| **流式接口** | SSE（Server-Sent Events），`Accept: text/event-stream` |

### 通用响应结构

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

| code | 含义 |
|------|------|
| 0 | 成功 |
| 1001 | 参数错误 |
| 1002 | 未登录 / Token 过期 |
| 1003 | 无权限 |
| 2001 | 数据不存在 |
| 2002 | 操作冲突（如重复领取剧本） |
| 3001 | AI 服务异常 |
| 5000 | 服务器内部错误 |

### 通用请求头

```
Content-Type: application/json
Authorization: Bearer <jwt_token>    // 除登录/注册外必传
```

### 3.2 AI NPC 对话（流式 SSE）

核心接口，与剧本中的 NPC 进行实时对话。

```
POST /play/{progressId}/chat
Accept: text/event-stream
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| npcId | string | 是 | 对话的 NPC ID |
| message | string | 是 | 游客发送的消息内容 |
| nodeId | string | 是 | 当前所在的剧情节点 ID |

**SSE 流式响应**：

```
event: message
data: {"text": "哈哈，这你算是"}

event: message
data: {"text": "问对人了。"}

event: message
data: {"text": "这村口的古樟树，据说有八百多年了……"}

event: task
data: {"type": "puzzle", "taskId": "task_003", "title": "破译族谱密码", "description": "老村长提到了族谱中的秘密，试着解读它"}

event: choice
data: {"options": [{"id": "opt_a", "label": "追问祠堂的事"}, {"id": "opt_b", "label": "先去古井看看"}]}

event: done
data: {"newClues": ["clue_001"]}
```

| event 类型 | 说明 |
|------------|------|
| `message` | 对话文本片段（累积拼接后展示） |
| `task` | AI 判断到达了触发任务的时机，推送新任务 |
| `choice` | AI 给出剧情分支选项，游客需做出选择 |
| `done` | 本轮对话结束，携带关系值变化和新获得的线索 |

---

```json
  "data": {
    "nextNodeId": "n_005",
    "message": "你决定跟随老村长前往祠堂……"
  }
}
```

## 五、AI 图片生成模块

> 主要供管理端使用——管理员在配置剧本场景图和 NPC 立绘时，通过以下接口调用 AI 自动生成图片。

### 5.1 生成场景图

根据场景描述，AI 生成剧本剧情节点的场景配图。

```
POST /ai/generate-scene-image
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| scriptId | string | 是 | 所属剧本 ID |
| description | string | 是 | 场景描述（如"古樟村村口的百年牌坊，黄昏时分，夕阳透过树叶洒下斑驳光影"） |
| style | string | 否 | 风格偏好：`realistic`（写实）/ `ink`（水墨）/ `illustration`（插画），默认 realistic |
| aspectRatio | string | 否 | 宽高比：`16:9` / `4:3` / `1:1`，默认 `16:9` |
| referenceImage | string | 否 | 参考图 URL（可选，用于图生图） |

> **图片尺寸约束**（智谱 CogView-4）：宽高需在 512-2880px 之间、均为 16 的整数倍、总像素 < 2²¹(2,097,152)。默认映射：16:9→1536×864, 4:3→1152×864, 1:1→1024×1024。

**响应**：
```json

  "data": {
    "taskId": "gen_scene_001",
    "status": "processing",
    "estimatedSeconds": 15
  }
}
```

### 5.4 确认使用/放弃生成结果

管理员预览图片后，确认使用（关联到剧本节点/NPC）或放弃（图片标记删除）。

```
POST /ai/confirm-image
```

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| taskId | string | 是 | 生成任务 ID |
| action | string | 是 | `accept`（确认使用）/ `reject`（放弃） |
| targetType | string | 是（accept 时） | `scene` / `npc` / `cover` |
| targetId | string | 是（accept 时） | 场景为 nodeId，NPC 为 npcId，封面为 scriptId |
| imageUrl | string | 是（accept 时） | 选中的图片 URL |

**响应**：
```json
{
  "code": 0,
  "data": {
    "action": "accept",
    "targetType": "scene",
    "targetId": "n_003",
    "updatedUrl": "https://oss.example.com/ai-generated/scene_s_001_n_003.jpg"
  }
}
```

> `accept`：后端将图片 URL 写入对应的 script_nodes.scene_image 或 script_npcs.avatar 字段。
> `reject`：删除 OSS 上的临时图片文件。

### 5.5 后端 AI 图片生成实现要点

```
接口 /ai/generate-xxx 处理流程：
  1. 接收参数，组装完整 Prompt
     ├── 正向词：用户描述 + 固定质量词（"high quality, 4k, detailed"）
     └── 负向词：固定（"blurry, watermark, text, signature, modern buildings, ugly"）
  2. 选择 AI 模型 API（按配置）
     ├── 方案 A：Stable Diffusion（Replicate / Stability AI 官方 API）
     ├── 方案 B：通义万相（阿里云，国内访问快，中文理解好）
     └── 方案 C：OpenAI DALL-E 3（质量高但贵）
  3. 异步提交生成任务，立即返回 taskId
  4. 轮询或 webhook 等待完成
  5. 下载生成图片 → 上传到 OSS → 更新 task 状态为 completed
  6. 等管理员调用 /ai/confirm-image 确认是否使用

通义万相 API 调用示例（Python）：
  import requests
  response = requests.post(
      "https://dashscope.aliyuncs.com/api/v1/services/aigc/text2image/image-synthesis",
      headers={"Authorization": f"Bearer {api_key}"},
      json={
          "model": "wanx2.0-t2i-turbo",
          "input": {
              "prompt": prompt,
              "negative_prompt": "blurry, watermark, text, low quality"
          },
          "parameters": {
              "size": "1536*864",
              "n": 1
          }
      }
  )
  # 同步返回结果，约 5-15 秒
```
